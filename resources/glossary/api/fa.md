---
term: API
---

مخفف "*Application Programming Interface*". در زمینه عمومی محاسبات، API مجموعه‌ای از قوانین و مشخصاتی است که نرم‌افزار می‌تواند برای ارتباط با یکدیگر از آن پیروی کند. آنها به توسعه‌دهندگان اجازه می‌دهند تا به قابلیت‌ها یا داده‌های یک برنامه، سیستم‌عامل یا سرویس دیگر برای نرم‌افزار خود دسترسی پیدا کنند.


> ► *به فارسی می‌توان آن را به عنوان "برنامه‌نویسی کاربردی Interface" یا مستقیماً به عنوان "برنامه‌نویسی Interface" ترجمه کرد.*
