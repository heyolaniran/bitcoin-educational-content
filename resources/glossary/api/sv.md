---
term: API
---

Akronym för "*Application Programming Interface*". I det allmänna datorsammanhanget är ett API en uppsättning regler och specifikationer som programvaror kan följa för att kommunicera med varandra. De gör det möjligt för utvecklare att få tillgång till funktioner eller data från en applikation, ett operativsystem eller en annan tjänst för sin egen programvara.


> ► *På engelska kan det översättas som "application programming Interface" eller direkt som "programming Interface".*
