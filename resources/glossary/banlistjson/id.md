---
term: BANLIST.JSON

---
Nama file yang digunakan oleh perangkat lunak Bitcoin Core untuk mencatat alamat IP _node_ yang telah dilarang oleh pengguna. File ini berisi daftar _node_ yang diblokir dan _node_ tersebut tidak akan terhubung. Fitur ini membantu mencegah interaksi dengan _node_ dengan potensi bahaya atau jahat.
