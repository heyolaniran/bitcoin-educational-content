---
term: BRANCH (GIT)
---

I samband med Git representerar en separation från huvudarbetsflödet, vilket möjliggör parallell utveckling utan att påverka huvudgrenen (vanligtvis kallad `master` eller `main`). Grenar gör det möjligt att utföra ändringar, tester och experiment i en isolerad miljö innan de integreras i huvudprojektet via en sammanslagning.
