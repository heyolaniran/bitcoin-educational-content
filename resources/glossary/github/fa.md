---
term: گیت‌هاب
---

پلتفرمی برای مدیریت و میزبانی کد منبع که همکاری بین توسعه‌دهندگان را تسهیل می‌کند. همان‌طور که از نامش پیداست، GitHub بر اساس سیستم کنترل نسخه Git ساخته شده است. این پلتفرم امکان پیگیری تغییرات کد، مدیریت نسخه‌ها و تشویق به همکاری از طریق ابزارهایی مانند درخواست‌های کشش و مسائل را فراهم می‌کند. GitHub به ابزاری ضروری برای توسعه‌دهندگان تبدیل شده است، به‌ویژه در جامعه Bitcoin که اکثر پروژه‌ها، از جمله Bitcoin Core، اجرای اصلی پروتکل، در آن میزبانی می‌شوند. در سال ۲۰۱۸، مایکروسافت GitHub را به مبلغ ۷.۵ میلیارد دلار خریداری کرد.
