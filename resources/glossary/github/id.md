---
term: GITHUB

---
Sebuah platform untuk mengelola dan _hosting_ kode sumber yang memfasilitasi kolaborasi di antara para pengembang. Seperti namanya, GitHub dibangun di atas sistem kontrol versi Git. Platform ini memungkinkan untuk melacak perubahan kode, mengelola versi, dan mendorong kolaborasi melalui alat bantu seperti _pull request_ dan _issue_. GitHub telah menjadi alat penting bagi para pengembang, terutama di dalam komunitas Bitcoin di mana sebagian besar proyek, termasuk Bitcoin Core, implementasi utama protokol, di-_hosting_. Pada tahun 2018, Microsoft mengakuisisi GitHub dengan nilai 7,5 miliar dolar.
