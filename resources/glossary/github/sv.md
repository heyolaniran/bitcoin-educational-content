---
term: GITHUB
---

En plattform för hantering och hosting av källkod som underlättar samarbete mellan utvecklare. Som namnet antyder bygger GitHub på versionskontrollsystemet Git. Denna plattform gör det möjligt att spåra kodändringar, hantera versioner och uppmuntra till samarbete genom verktyg som pull requests och issues. GitHub har blivit ett viktigt verktyg för utvecklare, särskilt inom Bitcoin-communityn där majoriteten av projekten, inklusive Bitcoin Core, den huvudsakliga implementeringen av protokollet, är värd. År 2018 förvärvade Microsoft GitHub för 7,5 miljarder dollar.
