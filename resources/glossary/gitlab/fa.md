---
term: گیت‌لب
---

یک پلتفرم مدیریت و میزبانی کد منبع که همکاری بین توسعه‌دهندگان را تسهیل می‌کند. GitLab جایگزین اصلی GitHub است. این پلتفرم بسیار مشابه است، اما همچنین امکان میزبانی خود را نیز ارائه می‌دهد. همانطور که از نامش پیداست، GitLab بر اساس سیستم کنترل نسخه Git ساخته شده است. این پلتفرم به این ترتیب امکان ردیابی تغییرات کد، مدیریت نسخه‌ها و تقویت همکاری از طریق ابزارهایی مانند درخواست‌های کشش و مسائل را فراهم می‌کند. برخی از پروژه‌های مرتبط با Bitcoin مانند Samourai Wallet، Whirlpool، یا RoninDojo از GitLab استفاده می‌کنند.
