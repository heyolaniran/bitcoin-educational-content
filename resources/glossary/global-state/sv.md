---
term: Global State
---

I RGB-protokollet betecknar Global State uppsättningen offentliga egenskaper för en Contract, som upprättats under Genesis. Det fungerar som ett offentligt register, som grupperar tillstånd Elements som är tillgängliga för alla och kan uppdateras av auktoriserade övergångar i enlighet med de regler som definieras av Contract Schema. Till skillnad från Owned States, som är associerade med specifika enheter, tillhör Global State inte någon särskild part.
