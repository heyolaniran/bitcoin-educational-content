---
term: GUI
---

Akronym för "*Graphical User Interface*", eller "Interface graphique utilisateur" på franska. Det är en typ av användare Interface som tillåter interaktion med programvara genom interaktiva visuella Elements (knappar, ikoner, bilder, fönster ...) och som föredrar att använda pekdon (musen) framför textkommandon som med CLI.
