---
term: Interface Implementation
---

در پروتکل RGB، Interface Implementation به مجموعه‌ای از اعلامیه‌ها اشاره دارد که ارتباط بین یک Interface و یک Schema را برقرار می‌کند. این امر ترجمه معنایی داده‌های باینری Contract را ممکن می‌سازد تا تفسیر صحیح آن‌ها توسط کاربر یا برنامه‌هایی مانند کیف پول‌ها تضمین شود.
