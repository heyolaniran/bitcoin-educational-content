---
term: Lightning Network
---

Ett högre Layer-protokoll som bygger på Bitcoin-protokollet och som syftar till att möjliggöra snabba och billiga transaktioner. Det gör det möjligt att skapa betalningskanaler mellan deltagare, inom vilka transaktioner kan göras nästan omedelbart och med minimala avgifter, utan att behöva registrera varje enskild transaktion på Blockchain. Kanalerna kan vara öppna nästan obegränsat och kräver endast transaktioner på Blockchain när de öppnas och stängs. Lightning Network syftar till att förbättra Bitcoin:s skalbarhet och göra det möjligt att genomföra betalningar av mindre värde.
