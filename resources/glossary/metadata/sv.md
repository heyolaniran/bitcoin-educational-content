---
term: METADATA
---

Inom det allmänna området databehandling avses data som ger information om andra data. De beskriver egenskaper, innehåll, kvalitet, format och struktur hos de data som de åtföljer. Man skiljer alltså mellan nyttolasten, som utgör kärnan i informationen, och metadata. För ett dokument kan metadata till exempel omfatta författare, skapandedatum, filstorlek och associerade nyckelord. I Bitcoin finns metadata i många Elements. Det används i mottagningsadresser, utökade nycklar, block osv.
