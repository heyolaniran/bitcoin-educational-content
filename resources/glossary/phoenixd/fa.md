---
term: PHOENIXD
---

اجرای تخصصی یک نود لایتنینگ (lightning)، طراحی شده برای ارسال و دریافت پرداخت‌ها از یک سرور. Phoenixd از همان نرم‌افزار Phoenix mobile Wallet/node استفاده می‌کند، اما برخلاف دومی که به‌طور انحصاری برای گوشی‌های هوشمند طراحی شده است، phoenixd قادر به اجرا بر روی یک سرور به عنوان daemon است. به جای داشتن یک رابط کاربری گرافیکی Interface (GUI)، این نرم‌افزار به یک HTTP API Interface مجهز شده و از مدیریت خودکار کانال و نقدینگی پشتیبانی می‌کند. Phoenixd یک پروژه توسعه یافته و نگهداری شده توسط شرکت فرانسوی Acinq است.
