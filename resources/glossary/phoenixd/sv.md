---
term: PHOENIXD
---

Specialiserad implementering av en Lightning (blixt)-nod, utformad för att skicka och ta emot betalningar från en server. Phoenixd använder samma programvara som Phoenix mobile Wallet/node, men till skillnad från den senare, som uteslutande är avsedd för smartphones, kan phoenixd köras på en server som en daemon. Istället för att ha ett grafiskt användargränssnitt Interface (GUI) är den utrustad med ett HTTP API Interface och stöder automatisk kanal- och likviditetshantering. Phoenixd är ett projekt som utvecklas och underhålls av det franska företaget Acinq.
