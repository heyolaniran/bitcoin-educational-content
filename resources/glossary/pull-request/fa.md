---
term: درخواست کشیدن
---

در زمینه GitHub و سایر پلتفرم‌های میزبانی کد، یک Pull Request درخواستی است که توسط یک مشارکت‌کننده برای ادغام تغییرات خود از یک شاخه از Fork خود به یک شاخه از مخزن اصلی انجام می‌شود. این درخواست یک بازبینی کد و بحث را قبل از اینکه تغییرات به‌طور بالقوه ادغام شوند، به جریان می‌اندازد. این فرآیند به‌طور گسترده‌ای در توسعه پیاده‌سازی‌های نود Bitcoin، به‌ویژه Bitcoin Core، استفاده می‌شود.


> ► *اصطلاح "درخواست کشش" اغلب به صورت "PR" مخفف می‌شود.*
