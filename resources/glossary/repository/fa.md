---
term: مخزن
---

یک ساختار داده مرکزی که در Git استفاده می‌شود و اطلاعات نسخه‌بندی یک پروژه در آن ذخیره می‌شود. یک مخزن شامل تاریخچه کامل تمامی تغییرات، شاخه‌ها و برچسب‌ها است. هر مخزن یک مجموعه مستقل از فایل‌ها و پوشه‌ها است که همراه با تاریخچه‌ای از تعهدات، امکان همکاری و پیگیری تغییرات در طول زمان را فراهم می‌کند. به عنوان مثال، مخزن Bitcoin Core در GitHub در اینجا میزبانی می‌شود: https://github.com/Bitcoin/Bitcoin.


> ► *در زبان انگلیسی به آن "مخزن" گفته می‌شود. استفاده از اختصار "repo" برای اشاره به یک مخزن Git رایج است.*
