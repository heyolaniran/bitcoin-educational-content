---
term: REPOSITORY
---

En central datastruktur som används i Git där versionsinformation för ett projekt lagras. Ett repository innehåller den fullständiga historiken för alla ändringar, grenar och taggar. Varje arkiv är en oberoende samling av filer och mappar som åtföljs av en historik över överföringar, vilket möjliggör samarbete och spårning av ändringar över tid. Till exempel finns Bitcoin Core-arkivet på GitHub här: https://github.com/Bitcoin/Bitcoin.


> ► * På engelska kallas det för "repository". Det är vanligt att använda förkortningen "repo" för att hänvisa till ett Git-repository.*
