---
term: STALE (BLOCK)
---

Avser ett block utan barn: ett giltigt block, men uteslutet från den huvudsakliga Bitcoin-kedjan. Det inträffar när två gruvarbetare hittar ett giltigt block på samma kedjehöjd inom en kort tidsperiod och sänder det över nätverket. Noder väljer så småningom ett enda block att inkludera i kedjan, baserat på principen om kedjan med mest ackumulerat arbete, vilket gör de andra "föräldralösa", "föråldrade" eller "inaktuella"
