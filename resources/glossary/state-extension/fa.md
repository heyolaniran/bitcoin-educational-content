---
term: State Extension
---

در زمینه پروتکل RGB، یک State Extension یک عملیات قراردادی است که برای آغاز به‌روزرسانی وضعیت یک Contract از طریق بازخرید Valencies اعلام شده قبلی استفاده می‌شود. این فرآیند از یک حق عمومی ثبت شده در Contract برای ایجاد یک تغییر استفاده می‌کند، که تنها زمانی مؤثر می‌شود که توسط یک State Transition بسته شود. مورد آخر شامل به‌روزرسانی نهایی Contract State است.
