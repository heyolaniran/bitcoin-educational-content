---
term: MAPPER
---

Dalam konteks ilmu komputer, _mapping_ (pemetaan) mengacu pada proses mengasosiasikan elemen dari satu kumpulan data dengan elemen dari kumpulan data lainnya dengan cara yang sistematis. Asosiasi ini memungkinkan data di set pertama untuk diganti dengan data di set kedua, atau untuk bertransisi dari satu set ke set lainnya. Teknik ini sering digunakan dalam operasi transformasi data.
